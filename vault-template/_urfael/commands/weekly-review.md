description: Sweep the inbox, file loose notes, surface stale projects, summarize the week
---

Run the user's weekly review. Be a librarian: organize, don't dump.

1. **Inbox sweep.** For each note in `00_Inbox/`: decide if it's a Project, Area, Resource,
   or Archive item; move it to the right PARA folder; fix frontmatter and `[[wikilinks]]`;
   update any backlinks if you rename. Propose the moves first if there's any ambiguity.
2. **Project health.** Scan `01_Projects/`. Flag projects marked `active` with no update or
   next action in 14+ days. Suggest archiving anything done.
3. **Open loops.** Reconcile `MEMORY.md` "Open loops" against reality; close what's done,
   surface what's stalled.
4. **Week summary.** From the last 7 daily notes in `90_Daily/`, write a tight summary of
   what happened, what shipped, and what's carrying over. Save it under `## Done`/a weekly
   note as appropriate.
5. **Memory update.** Update `MEMORY.md` with anything durable you learned. Bump "Last updated".
6. Report a short spoken-style summary: what you filed, what's stale, what carries over.
